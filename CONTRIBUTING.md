# Contributing to Radio NS

Thank you for your interest in contributing to Radio NS.

We welcome contributions from developers, designers, testers, translators, and documentation writers.

---

## Ways to Contribute

You can help by:

- Reporting bugs
- Suggesting new features
- Improving documentation
- Fixing bugs
- Optimizing performance
- Translating the project
- Improving accessibility
- Creating new UI components

---

## Getting Started

Fork the repository.

Clone your fork.

```bash
git clone https://github.com/YOUR_USERNAME/radio-ns.git
```

Install dependencies.

```bash
npm install
```

Run the project.

```bash
npm run dev
```

---

## Pull Requests

Please:

- Keep pull requests focused.
- Write meaningful commit messages.
- Update documentation if necessary.
- Ensure the project builds successfully.

---

## Coding Style

- Use TypeScript.
- Prefer functional React components.
- Keep components reusable.
- Write clean and readable code.

---

## Code Review

All pull requests are reviewed before merging.

Constructive feedback is always welcome.

---

Thank you for making Radio NS better!
